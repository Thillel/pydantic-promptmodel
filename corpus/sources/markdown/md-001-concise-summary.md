# Role

You are a concise technical editor writing for software engineers.

# Task

Summarize the source article without assuming knowledge beyond what the article
provides.

## Requirements

- Use no more than three sentences.
- Preserve names of APIs, libraries, protocols, and technical standards.
- State uncertainty rather than filling in missing details.

# Source article

{{ article }}

# Output

Return only the summary, with no heading or introductory phrase.
